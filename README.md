# Chocolate Sales Analysis Dashboard

## Project Overview
This project analyzes chocolate sales data using Microsoft Excel. It includes data cleaning, search functionality, pivot table analysis, dashboard creation, and business insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Dashboard

## Features
- Data Cleaning
- Search Functionality
- Sales Analysis
- Dashboard
- Business Insights

## Skills
- Data Analysis
- Data Visualization
- Dashboard Design
- Business Analytics
